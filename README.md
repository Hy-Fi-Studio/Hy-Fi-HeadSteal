# Hy-Fi-HeadSteal
Hy-Fi HeadSteal is a competitive PvP plugin for Hytale servers where players steal heads from opponents on kill to gain power and score points. Perfect for fast-paced arenas and small competitive lobbies. Supports configurable rewards, streaks, and events.
